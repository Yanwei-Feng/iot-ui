# IOT UI 组件库
在线预览：[戳我 😘](https://github.com/Yanwei-Feng/iot-ui)

## 开发
 * 提交代码
 ```bash
   yarn commit
 ```
  使用 yarn commit 替代 git commit生成规范的 Commit Message，当然为了效率你可以选择手写，但是要符合规范

本地预览：

```bash
git clone https://github.com/Yanwei-Feng/iot-ui.git
cd iot-ui
yarn
yarn start
```

按顺序执行完命令后，即可在 localhost:3000 端口看到以下内容：
